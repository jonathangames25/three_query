Prototyping Guide & Cheatsheet

1. Built-in Animation Loops (No Scripting Needed)

Animate translations and rotations continuously inside the Three.js loop by specifying speeds and directions on spawn or modify.

  • Continuous Rotation: modify name=mybox rotateSpeed=90 rotateDirection=0,1,0 (Rotates on Y axis at 90 degrees/sec)
  • Continuous Movement: modify name=mybox moveSpeed=5 moveDirection=0,0,1 (Moves forward along Z axis at 5 units/sec)
  • Rotation Axis Shortcuts: Directions can be specified as axis values like 1,0,0, or shortcut strings: x, y, z, up, down, left, right, forward, back.

2. Input-Driven Animations (No Scripting Needed)

Trigger movements or rotations only when specific keys are held by passing the key name to moveInput or rotateInput. Key values include standard letters (w, a, s, d) or arrows (up_arrow, down_arrow, left_arrow, right_arrow, space).

  • Move left on ArrowLeft: modify name=ship moveSpeed=10 moveDirection=left moveInput=left_arrow
  • Move right on ArrowRight: modify name=ship moveSpeed=10 moveDirection=right moveInput=right_arrow
  • Continuous rotation when pressing Space: modify name=wheel rotateSpeed=180 rotateDirection=z rotateInput=space

3. Grouping and Hierarchies (Transforms)

Use transform types to create pivots/groups. Linking child meshes to parent transforms allows you to move, rotate, or scale the entire group at once.

  • Create Pivot: spawn type=transform name=pivot position=0,3,0
  • Link meshes: link child=orbiter1 parent=pivot
  • Orbit all children: modify name=pivot rotateSpeed=60 rotateDirection=y

4. Physics & Collisions

Add physics rigid bodies and colliders to objects to make them fall, bounce, and collide.

  • Spawn Dynamic Box: spawn type=mesh shape=box name=crate position=0,10,0
  • Add Dynamic Physics: spawn type=collision target=crate shape=box static=false
  • Spawn Static Ground: spawn type=mesh shape=plane name=floor position=0,0,0 scale=50,1,50
  • Add Static Physics: spawn type=collision target=floor shape=box static=true

5. Character Controller

Spawn an active First Person Controller to explore the scene manually with WASD + Mouse.

  • Spawn FPC: spawn type=character name=hero position=0,2,5

6. Dynamic Skybox & Environment

  • Change Skybox & Fog Color: modify name=skybox color=#020617 or spawn type=skybox color=#020617

7. Command Syntax Reference

  • Spawn: spawn type=[mesh|light|transform|skybox|collision|character|script] name=[name] [args...]
  • Modify: modify name=[name] position=[x,y,z] rotation=[x,y,z] scale=[x,y,z] color=[hex] intensity=[val]
  • Delete: delete name=[name]
  • Link: link child=[name] parent=[name]
  • Get: get name=[name|all]
